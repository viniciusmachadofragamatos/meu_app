Como o objetivo é aprender, tem paginas que foram feitas para aprendizado e nao com o intuito de deixar o aplicativo em padrão para usuario, como por exemplo:
as paginas de lista horizontal e lista vertical, as configurações com perguntas sem nexo e os dados cadastrais, onde testamos componentes como RadioButton, Switch button e etc, usamos 3 formas de armazenar dados diferentes que foram shared_preferences, Banco de dados Hive e Banco de dados SQLite apenas para aprendizado, e deixamos 2 "soltos" sem usar.
